# PHP example project

This project is a simple example for GitLab CI with PHP environment.


## Usage

```bash
docker login registry.gitlab.com
docker run -d -p 80:80 registry.gitlab.com/geoffrey-grebert/php-test
curl 127.0.0.1
```
